Translator

Computer

Traduction

  • Computer: An electronic machine capable of storing information and automatically processing it through mathematical and logical operations controlled by software.

  • Compatible Computer: A computer capable of running the same programs as other computers of the same type.

  • Personal Computer: A desktop computer that includes a central unit, keyboard, monitor, and one or more disk drives, which can operate without being connected to a computer network.

  • Portable Computer: A personal computer with a weight and size so reduced that it can be comfortably carried from one place to another, as if it were a briefcase; it includes a built-in battery that allows for a certain degree of autonomous operation.
